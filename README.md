Titanic Survival Analysis and Prediction
This is a two-part data science project built on the Titanic dataset. Part 1 focused on exploratory data analysis — understanding who was on board, how survival varied across gender, class, fare, and age, and what the data revealed through visualisations. Part 2 took those insights further by building a machine learning classification model to predict survival.

What's Inside
The project covers the full pipeline — data cleaning, feature engineering, train-test splitting, training three classifiers (Logistic Regression, Decision Tree, and Random Forest), and evaluating the best model using a confusion matrix, classification report, and feature importance scores.

Result
Random Forest performed best with 77.53% accuracy. Sex, fare, and age turned out to be the strongest predictors of survival — which aligned perfectly with what the EDA had already suggested.

Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, and Scikit-learn.
